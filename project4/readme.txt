---Menu Instructions---
Select a data structure from the main menu. From there select a operation type, 
(access, mutate, or traverse). 

Build From File is found in the mutate menu. The example text files included 
are: 

Graph: graph.txt
Digraph: digraph.txt

After building the data structure from the respective text file, you may perform operations on it.
